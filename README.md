# IA Driven Development Resources

Este repositorio reúne **demos, plantillas y recursos para el desarrollo impulsado por IA**. Está diseñado para desarrolladores, investigadores y entusiastas que quieran experimentar con **agents, prompts, skills y más**, facilitando un flujo de trabajo rápido y replicable en proyectos de IA.

## Contenido del repositorio

* **Agents** – Ejemplos de agentes de IA configurables y listos para usar.
* **Prompts** – Plantillas y ejemplos de prompts optimizados para distintos casos de uso.
* **Skills** – Habilidades predefinidas que se pueden integrar en agentes para extender su funcionalidad.
* **Demos** – Implementaciones de referencia que muestran cómo combinar prompts, agentes y skills.
* **Templates** – Archivos base para acelerar la creación de nuevos agentes, prompts o skills.

## Cómo usarlo

1. Clona el repositorio:

```bash
git clone https://github.com/tu-usuario/ia-driven-resources.git
```

2. Explora la carpeta correspondiente al recurso que quieras usar (`agents/`, `prompts/`, `skills/`, `demos/`).

3. Personaliza las plantillas según tus necesidades de desarrollo.

4. Ejecuta los demos para ver cómo interactúan los agentes con los prompts y skills.
